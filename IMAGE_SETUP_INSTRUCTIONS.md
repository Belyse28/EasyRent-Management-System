# Adding Real House Images to Your Property Management System

## Step 1: Download Real House Images

Download these 4 house images and save them in the `public/images/` folder:

### 1. Modern Apartment (apartment.jpg)
- Search for: "modern apartment building exterior"
- Recommended size: 400x300px
- Save as: `public/images/apartment.jpg`

### 2. Cozy House (house.jpg)
- Search for: "cozy suburban house exterior"
- Recommended size: 400x300px
- Save as: `public/images/house.jpg`

### 3. Studio Apartment (studio.jpg)
- Search for: "small studio apartment building"
- Recommended size: 400x300px
- Save as: `public/images/studio.jpg`

### 4. Family Home (family-home.jpg)
- Search for: "large family house with garage"
- Recommended size: 400x300px
- Save as: `public/images/family-home.jpg`

## Step 2: Image Sources (Free to Use)

You can download free house images from:
- **Unsplash.com** - Search "house exterior", "apartment building"
- **Pexels.com** - Search "real estate", "house"
- **Pixabay.com** - Search "house", "apartment"

## Step 3: Recommended Image Specifications

- **Format**: JPG or PNG
- **Size**: 400x300 pixels (4:3 aspect ratio)
- **Quality**: High resolution but compressed for web
- **File size**: Under 200KB each

## Step 4: After Adding Images

Once you've added the real images, the system will automatically use them instead of the SVG placeholders.

## Alternative: Use Online Images

If you prefer, you can use direct URLs to online images by updating the API service.
